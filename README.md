# Library Management System in Java

A simple console-based Library Management System developed using Java.  
This project allows users to add books, issue books, return books, and display available books.



## Features

- Add new books
- Display all books
- Issue books
- Return books
- Check available books


## Concepts Used

- Classes & Objects
- ArrayList (Collections)
- Conditional Statements
- Loops
- Methods



## Technologies Used

- Java
- IntelliJ IDEA



## Project Structure

LibraryManagementSystem.java


## How to Run

1. Open project in IntelliJ IDEA
2. Compile the program
3. Run `LibraryManagementSystem.java`
4. Use menu options to manage books


## Sample Menu

===== Library Management System =====

1. Add Book
2. Show Books
3. Issue Book
4. Return Book
5. Exit
